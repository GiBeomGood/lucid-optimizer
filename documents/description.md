# 프로젝트 목표

아이템 정보를 저장/관리하고, 최적의 아이템 조합을 찾아 제공하는 프로그램.

---

## 기능 1: JSON 파일 기반 최적 조합 탐색

- 아이템 정보가 저장된 JSON 파일을 읽어, 가능한 조합 중 최고 점수의 top-k를 반환
- 조합 생성 → 각 조합 점수 산출 → top-k 정렬 순서로 처리

## 기능 2: 아이템 관리 TUI CLI

- 터미널 UI로 아이템 추가/삭제/편집
- 화살표키 또는 `j`/`k`로 목록 탐색, Enter로 값 입력
- 변경 사항은 메모리에 먼저 저장되고, 저장 명령 시 JSON 파일에 반영

---

## JSON 파일 형식

아이템 옵션은 아래 6가지 중 하나이며, 각 옵션은 정수 값을 가진다. 아이템 하나당 옵션은 2개.

- `마력`, `마력%`, `숙련도`, `크리티컬 확률`, `크리티컬 데미지`, `재사용 대기시간 감소`

```json
{
  "크리티컬 데미지": 8,
  "크리티컬 확률": 13
}
```

---

## 화면 구성

**아이템 목록 화면**

```
  1. <option 1>: <num>
     <option 2>: <num>
> 2. <option 1>: <num>
     <option 2>: <num>
  ...
  아이템 추가하기
```

- 커서 위치에서 좌우 방향키로 값(`<num>`) 간 이동 및 수정
- JSON이 비어있으면 아이템 추가 안내 메시지 표시
- 화면 하단에 회색 힌트 텍스트 표시 (e.g., `Enter: 값 입력 | D: 삭제`)

**아이템 추가 흐름**

옵션 선택 → 값 입력 → 옵션 선택 → 값 입력 → 확정

- 추가 중 취소 가능
