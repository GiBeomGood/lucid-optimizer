## 변경 동기

- 화면 곳곳에 `아이템`이라는 표현이 남아 있어, 실제 게임 내 명칭인 `몽환의 결정`과 일치하지 않았음
- 힌트 텍스트에서 `o: 옵션`은 `v: 값 편집`과 표현 방식이 달라 일관성이 없었음
- 종료 확인 팝업의 안내 텍스트가 세로 기준으로 위쪽에 치우쳐 있었음

## 수정 내용

- **`아이템` → `몽환의 결정` 전면 교체** — 목록 타이틀, 빈 목록 안내 메시지, 추가 팝업 제목, 홈 메뉴 항목 모두 변경
- **힌트 텍스트 일관성 개선** — `o: 옵션` → `o: 옵션 편집`으로 변경하여 `v: 값 편집`과 표현 방식을 맞춤
- **종료 확인 팝업 세로 중앙 정렬** — 안내 텍스트가 팝업 내부 영역의 세로 중앙에 오도록 수정
